================================================================================
ニュートリノの物理学
================================================================================

**ニュートリノ** は **電荷を持たないレプトンの総称** である。
1900年代はじめ **原子核のベータ崩壊** の研究をしていたら、エネルギー保存則に

この問題を解決する案として、1930年にヴォルフガング・パウリ（Wolfgang Ernst Pauli）が
**ニュートリノ仮説** を発表した。**ニュートリノ** と名付けたのは、この仮説を定式化したエンリコ・フェルミ（Enrico Fermi）である）
