相対性理論
================================================================================

20世紀最大の物理学者とも呼ばれるアルバート・アインシュタイン（Albert Einstein）が完成させた理論体系。
1905年に **特殊相対性理論** 、1915年に **一般相対性理論** が発表された。

特殊相対性理論は、異なる慣性系（＝等速運動している系）

一般相対性理論は、重力の効果（＝加速度運動している系）
