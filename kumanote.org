* 加速器のビームパワー

   | 項目                       | J-PARC |    NuMI | CNGS    |
   |----------------------------+--------+---------+---------|
   | (1) バンチあたりの陽子数   | 1.5e13 |         |         |
   | (2) スピルあたりのバンチ数 |      8 |     420 |         |
   | (3) スピルあたりの陽子数   | 1.2e14 |    4e13 |         |
   | (4) 陽子のエネルギー       | 30 GeV | 120 GeV | 400 GeV |
   | (5) スピル間隔             |   2.48 |    1.87 |         |
   | (6) ビームパワー           | 240 kW |     404 |         |

   - CNGSのパラメータは分からなかった

** ビームパワーの計算
   - 主要素 : [ビームパワー] = [スピルあたりの陽子数] x [陽子のエネルギー] / [スピル間隔]
   - 具体的 : (6) = (3) x (4) / (5) x 1.6e-19 x 1e-3 （eV -> J, W -> kW 変換）

** NuMIビーム構造
   - スピル > バッチ > バンチ
   - バッチあたり84バンチ
   - スピルあたり (2+5) バッチ
     - 最初の2バッチ : Tevatron へ
     - 後ろの5バッチ : NuMI へ （5 x 84 = 420バンチ／スピル）
